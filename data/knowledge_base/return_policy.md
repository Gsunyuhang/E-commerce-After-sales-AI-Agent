# 退换货政策

## 一、7天无理由退货规则

### 1.1 适用条件
- 自商品签收之日起7天内（以物流签收时间为准），消费者可申请无理由退货。
- 商品须保持原状，不影响二次销售（含原包装、吊牌、配件、说明书等完整）。
- 商品无使用痕迹、无洗涤、无人为损坏。

### 1.2 退货流程
1. 消费者在订单页面或联系客服发起退货申请。
2. 客服审核退货资格（签收时间、商品状态等）。
3. 审核通过后，消费者将商品寄回指定退货地址。
4. 仓库收到退货商品并验收合格后，3-5个工作日内原路退回货款。

### 1.3 退款金额计算
- 7天无理由退货：退回商品实际支付金额，运费按运费规则处理。
- 若购买时享受满减优惠，退货后不满足优惠条件的，需扣除相应差价。

## 二、换货规则

### 2.1 适用条件
- 自商品签收之日起7天内，因商品质量问题可申请换货。
- 换货商品须为同款同价商品的不同规格（如尺码、颜色）。
- 非质量问题换货，运费由消费者承担。

### 2.2 换货流程
1. 消费者提交换货申请，说明换货原因及目标规格。
2. 客服核实订单信息及商品状态。
3. 消费者寄回原商品，仓库验收后发出新商品。
4. 换货处理周期一般为5-7个工作日。

### 2.3 商品质量问题换货
- 商品存在质量问题（如破损、功能故障、与描述严重不符），消费者可申请换货。
- 质量问题换货的来回运费均由商家承担。
- 超出7天但在质保期内的，按质保政策处理。

## 三、不可退货商品类目

以下商品不支持7天无理由退货：

1. **贴身衣物**：内衣、内裤、泳装等（卫生安全原因）。
2. **美妆个护**：已拆封的化妆品、护肤品（卫生安全原因）。
3. **食品类**：生鲜食品、已拆封零食等（保质期原因）。
4. **定制商品**：个性化定制商品（已开始生产的）。
5. **虚拟商品**：充值卡、会员服务等（已使用的）。
6. **已安装商品**：已安装或已使用的大家电（需联系品牌售后）。
7. **书籍音像**：已拆封的书籍、光盘等。

## 四、特殊场景处理

### 4.1 签收超期
- 超过7天但未超过15天：不支持无理由退货，但可协商换货或维修。
- 超过15天：按质保政策处理，联系品牌售后。

### 4.2 商品损坏责任认定
- 签收时商品已损坏：消费者需在签收后48小时内拍照举证，商家承担全部责任。
- 签收后损坏：需根据损坏原因判定责任，人为损坏不支持退换货。

### 4.3 部分退货
- 订单含多件商品时，支持部分商品退货。
- 部分退货后不满足满减/包邮条件的，需补退差价。
