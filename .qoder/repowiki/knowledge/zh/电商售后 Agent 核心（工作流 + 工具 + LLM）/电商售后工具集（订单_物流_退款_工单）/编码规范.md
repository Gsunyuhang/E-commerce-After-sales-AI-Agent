- 每个工具类在 `__init__` 中调用 `super().__init__(name=..., description=...)` 声明工具标识与面向用户的描述，供 LangChain 暴露。
- 参数定义采用 JSON Schema 风格：`get_parameters_schema()` 返回含 `type/object`、`properties`、`required` 的字典，并在 `validate_params` 中做等价的运行时校验。
- `_execute` 内部使用 `with get_db_session() as session:` 上下文管理数据库连接，查询失败时返回 `{found: False, message: ...}` 而非抛异常。
- 所有外部 I/O 异常在 `_execute` 末尾统一捕获并 `raise DatabaseError(...)`，交由基类 `execute()` 转换为带 `fallback` 的标准错误响应。
- 对外返回结果统一包含 `tool_name` 字段，成功路径附加 `data`，失败路径附加 `error`，便于上层聚合统计。
