- 配置项统一在 Settings 中以 pydantic field 声明默认值，并通过 @property 派生绝对路径或连接串，避免调用方拼接路径。
- 全局可复用对象（Settings 实例）一律通过 @lru_cache() 装饰的工厂函数 get_settings() 获取，禁止直接 new。
- LLM Prompt 以模块级大写常量字符串集中存放，使用 Python str.format 占位符 {xxx} 注入变量，并由 format_prompt 统一包装校验。
- 自定义异常继承自 AgentBaseError，并在构造时附带面向用户的 fallback_response，上层通过 handle_agent_error 统一转换为降级话术。
- 日志记录通过 log_chat_turn / log_tool_call 等高层封装写入，结构化字段统一放入 extra_data dict 再经 JsonFormatter 序列化。
